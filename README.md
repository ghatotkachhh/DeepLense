# DeepLense
![DEEPLENSE (2)](https://github.com/user-attachments/assets/dea97e0b-1ccb-4490-ac72-3536fb2a48b2)

DeepLense is a deep learning framework dedicated to searching for particle dark matter through strong gravitational lensing. It continuously experiments with strong gravitational lensing images and machine learning techniques to integrate these approaches for understanding dark matter morphology. Projects under DeepLense include super-resolution, physics-informed neural networks, self-supervised learning, and more.

This repository contains various projects under the DeepLense initiative. Below is a summary of the projects.
